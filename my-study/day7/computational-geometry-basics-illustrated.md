## 1. 欧几里得空间

### 什么是欧几里得空间？

**通俗解释**：欧几里得空间就是我们日常生活中理解的"空间"。

- **一维空间 E¹**：一条直线（只有前后）
- **二维空间 E²**：一个平面（有前后、左右）
- **三维空间 E³**：我们生活的世界（有前后、左右、上下）
- **d维空间 Eᵈ**：数学上的推广，有d个方向

### 图解

```
一维空间 E¹ (一条线)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━→ x
        ↑
      这里只有一个坐标 x

二维空间 E² (一个平面)
        ↑ y
        │
        │    ● p = (x, y)
        │   ╱
        │  ╱
        │ ╱
        │╱
━━━━━━━━┼━━━━━━━━━━━━━━━━━━━━━━━→ x
        │
      这里有两个坐标 (x, y)

三维空间 E³ (立体空间)
            ↑ z
            │
            │     ● p = (x, y, z)
            │    ╱│
            │   ╱ │
            │  ╱  │
            │ ╱   │
            │╱____│________→ y
           ╱
          ╱
         ╱
        ↙ x
      这里有三个坐标 (x, y, z)
```

### 点的表示

在d维空间中，一个点用d个数字来表示：

```
p = (x₁, x₂, ..., xd)

例如：
• 二维点：p = (3, 5) 表示向右走3步，向上走5步
• 三维点：p = (1, 2, 4) 表示向右1步，向前2步，向上4步
```

## 2. 直线与线段

### 2.1 直线的数学表示

**公式**：`αp₁ + (1-α)p₂`，其中 α 是任意实数

**通俗解释**：想象你站在点 p₁，面朝点 p₂。当你改变 α 的值：
- α = 1 时，你在 p₁ 的位置
- α = 0 时，你在 p₂ 的位置
- α 在 0 到 1 之间时，你在两点之间
- α > 1 时，你在 p₁ 的后面
- α < 0 时，你在 p₂ 的后面

### 图解：直线

```
                    α > 1           α = 1         0 < α < 1        α = 0          α < 0
                      ↓               ↓               ↓               ↓               ↓
━━━━━━━━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━●━━━━━━━━━━━━→
                   这里是            p₁           中间某处           p₂            这里是
                  p₁的延长                                                        p₂的延长

直线：所有可能的α值（从负无穷到正无穷）组成的点的集合
```

### 2.2 线段的数学表示

**公式**：`αp₁ + (1-α)p₂`，其中 **0 ≤ α ≤ 1**

**通俗解释**：线段就是直线的一部分，只包含两个端点之间的点。

### 图解：线段

```
                                    ┌─────────────────────────────┐
                                    │      这就是线段 p₁p₂        │
                                    └─────────────────────────────┘
                                               ↓
━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━●━━━━━━━━━━━━━━━━━━━━━━━━━━━━━●━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━ ━→
      直线的这部分不属于线段        p₁          线段部分          p₂    直线的这部分也不属于线段
                                  (α=1)      (0<α<1)          (α=0)

线段：只有 α 在 0 到 1 之间的点
```

### 2.3 具体例子

```
假设 p₁ = (0, 0)，p₂ = (10, 0)

当 α = 1 时：   1×(0,0) + 0×(10,0) = (0, 0)   → 在 p₁
当 α = 0.5 时： 0.5×(0,0) + 0.5×(10,0) = (5, 0) → 在中点
当 α = 0 时：   0×(0,0) + 1×(10,0) = (10, 0) → 在 p₂

图示：
    p₁                                           p₂
    ●━━━━━━━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━━━━━━━●
  (0,0)               (5,0)               (10,0)
   α=1                α=0.5                 α=0
```

## 4. 凸组合

### 什么是凸组合？

**公式**：∑αᵢpᵢ，其中 αᵢ ≥ 0 且 ∑αᵢ = 1

**通俗解释**：凸组合就是多个点的"加权平均"，权重都是非负的，且加起来等于1。

### 两个点的凸组合 → 线段

```
p₁ 和 p₂ 的所有凸组合形成它们之间的线段

    α₁ = 1, α₂ = 0          α₁ = 0.5, α₂ = 0.5          α₁ = 0, α₂ = 1
           ↓                        ↓                         ↓
           ●━━━━━━━━━━━━━━━━━━━━━━━━●━━━━━━━━━━━━━━━━━━━━━━━━●
          p₁                      中点                       p₂

计算：α₁p₁ + α₂p₂ = α₁p₁ + (1-α₁)p₂
```

### 三个点的凸组合 → 三角形

```
三个点 p₁、p₂、p₃ 的所有凸组合形成一个三角形

                    p₁ (α₁=1, α₂=0, α₃=0)
                    ╱╲
                   ╱  ╲
                  ╱    ╲
                 ╱  ●   ╲   ← 内部某点：α₁=0.3, α₂=0.3, α₃=0.4
                ╱        ╲     （0.3×p₁ + 0.3×p₂ + 0.4×p₃）
               ╱    ●     ╲
              ╱   重心      ╲  ← 重心：α₁=α₂=α₃=1/3
             ╱              ╲
            ╱________________╲
          p₂                  p₃
    (α₁=0,α₂=1,α₃=0)    (α₁=0,α₂=0,α₃=1)

三角形内的每一个点都可以表示为三个顶点的凸组合！
```

### 四个点的凸组合 → 四边形或四面体

```
在二维平面上：                          在三维空间中：
四个点的凸组合形成四边形                四个点的凸组合形成四面体

    p₁─────────────p₂                        p₁
     │╲           ╱│                         ╱│╲
     │ ╲         ╱ │                        ╱ │ ╲
     │  ╲       ╱  │                       ╱  │  ╲
     │   ╲     ╱   │                      ╱   │   ╲

                                              │
```

## 5. 多边形

### 什么是多边形？

**定义**：由有限条线段首尾相连围成的封闭图形。

**组成要素**：
- **顶点**：线段的端点
- **边**：连接相邻顶点的线段

### 多边形的严格定义

```
多边形由线段集合 L 定义，需满足：
1. L 中每条线段的每个端点，恰好是 L 中两条线段的公共端点
2. L 中任意两条线段上的点都可以通过 L 中的线段连通

简单说：边连成一个闭环，不能有断开的地方
```

## 7. 星形多边形

### 什么是星形多边形？

**定义**：在简单多边形 P 中，如果存在一个点 z，使得从 z 到 P 内任意点 p 的连线都完全在 P 内部，则称 P 为星形多边形。

**通俗理解**：站在某个位置能"看到"整个多边形内部的所有地方。

### 星形多边形的核

**定义**：所有满足上述条件的点 z 的集合称为"核"(kernel)。

### 8.2 平面直线图 (PSLG)

**定义**：所有边都是直线段的平面图。

### 8.3 欧拉公式（超级重要！）

**定理 1.1**：对于平面图，有

```
v - e + f = 2
```

其中：
- **v** = 顶点数 (vertices)
- **e** = 边数 (edges)
- **f** = 区域数 (faces)，包括外部无界区域

### 8.4.1 为什么选择对区域数 f 进行归纳？（思维过程揭秘）

这是一个很好的问题！初学者往往困惑：为什么不对顶点数 v 或边数 e 归纳，偏偏选择 f？

#### 核心洞察：找一个"容易控制变化"的量

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                    三 个 变 量 的 对 比 分 析                                  │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  变量    │ 如果对它归纳，需要做什么？          │ 难度评估                       │
│ ─────────┼────────────────────────────────────┼───────────────────────────────│
│    v     │ 删除一个顶点                        │ 😰 很难！                      │
│ (顶点数) │ 问题：删除顶点会同时删掉多条边      │ v减1，但e和f变化不确定         │
│          │ 而且可能把图弄得不连通              │                                │
│ ─────────┼────────────────────────────────────┼───────────────────────────────│
│    e     │ 删除一条边                          │ 😐 有点难                      │
│ (边数)   │ 问题：删除边后，f可能减1或不变      │ e减1，但f变化要分情况讨论      │
│          │ （取决于边是否在环上）              │                                │
│ ─────────┼────────────────────────────────────┼───────────────────────────────│
│    f     │ 删除环上的一条边                    │ 😊 最简单！                    │
│ (区域数) │ 做法：每删一条环边，正好少一个区域  │ f减1时，e也正好减1            │
│          │ 变化关系清晰、可控                  │ v不变，关系简单明确            │
│                                                                                │
└────────────────────────────────────────────────────────────────────────────────┘
```

#### 生活比喻：拆墙合并房间

```
想象你有一套房子，里面有很多房间（区域f）：

┌─────┬─────┬─────┐
│     │     │     │
│ 房1 │ 房2 │ 房3 │
│     │     │     │
├─────┼─────┼─────┤       "拆掉一面墙（删除一条边）
│     │     │     │        正好合并两个房间为一个"
│ 房4 │ 房5 │ 房6 │
│     │     │     │        墙（边）减少 1
└─────┴─────┴─────┘        房间（区域）减少 1
                           房角（顶点）不变！

拆掉房2和房5之间的墙：

┌─────┬─────┬─────┐
│     │     │     │
│ 房1 │     │ 房3 │
│     │ 房  │     │        现在只有5个房间了
├─────┤ 2+5 ├─────┤
│     │     │     │        e: 减少了1（一面墙没了）
│ 房4 │     │ 房6 │        f: 减少了1（两房合一）
│     │     │     │        v: 不变（墙角还在）
└─────┴─────┴─────┘

这就是为什么对 f 归纳最自然！
```

#### 数学上的精确解释

```
为什么 f 是最佳选择？

1. 当 f = 1 时（只有外部区域），图必定是"树"
   └─ 树有一个漂亮的性质：e = v - 1
   └─ 代入：v - (v-1) + 1 = 2 ✓ 基础情况成立！

2. 当 f > 1 时，图中必定存在"环"（cycle）
   └─ 环围成的就是一个区域
   └─ 删除环上任意一条边：
      • 顶点数 v 不变（没删顶点）
      • 边数 e 变成 e-1（删了一条边）
      • 区域数 f 变成 f-1（两个区域合并）

3. 变化量的关系：
   Δv = 0,  Δe = -1,  Δf = -1

   验证：Δv - Δe + Δf = 0 - (-1) + (-1) = 0

   如果小规模成立 v-(e-1)+(f-1)=2
   那么大规模也成立 v-e+f=2 ✓

这种"精确对应"的变化关系，让归纳证明变得优雅简洁！
```

#### 如果选择对 v 或 e 归纳会怎样？

```
❌ 对 v（顶点数）归纳的困难：

删除一个顶点时：
    ●────●────●            ●    ●────●
    │    │    │    删除    │         │
    │    │    │   中间  →  │         │
    ●────●────●   顶点     ●─────────●

v: 9 → 8 （减1，OK）
e: 12 → 8 （减了4条！不是减1）
f: 变化不确定...

太复杂了！每次删除顶点，e和f的变化都不一样，无法建立统一的递推关系。


❌ 对 e（边数）归纳的困难：

删除一条边时，有两种情况：

情况A：边在环上           情况B：边不在环上（桥）
●────●────●              ●────●    ●
│    │    │              │    │    │
│    │ ←删 │      vs     │    │←删 │
●────●────●              ●────●    ●

e: 减1                    e: 减1
f: 减1                    f: 不变！（图断开了）

两种情况f的变化不同，需要分类讨论，证明变复杂。
```

#### 总结：为什么 f 是最佳选择

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                                                                                │
│   选择对 f 归纳的三大理由：                                                    │
│                                                                                │
│   1️⃣ 基础情况简单：f=1 意味着图是树，有明确的 e=v-1 关系                      │
│                                                                                │
│   2️⃣ 归纳步骤优雅：删除环边时，v不变、e减1、f减1，变化精确可控                │
│                                                                                │
│   3️⃣ 物理直觉清晰：就像拆墙合并房间，操作简单，结果明确                       │
│                                                                                │
│   这种选择体现了数学证明中的智慧：                                             │
│   找到那个"变化最可控"的量进行归纳！                                          │
│                                                                                │
└────────────────────────────────────────────────────────────────────────────────┘
```

### 8.5 欧拉公式的推论

从欧拉公式可以推导出一些重要不等式。这些推论看起来复杂，但证明其实很简单！

#### 8.5.1 关键前提：握手定理（Handshaking Lemma）

在开始证明之前，我们需要一个重要工具——"握手定理"。

```

│  │  逻辑：如果有 f 个区域，每个区域至少3条边                               │  │
│  │        那么所有区域的边数加起来，至少是 3×f = 3f                        │  │
│  │                                                                         │  │
│  │        即：∑边数(f) ≥ 3f                                               │  │
│  └─────────────────────────────────────────────────────────────────────────┘  │
│                                                                                │
└────────────────────────────────────────────────────────────────────────────────┘
```

**🔑 Step 2 详解：为什么 ∑边数(f) = 2e ？（面握手定理）**

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                    理 解 " 每 条 边 被 两 个 区 域 共 享 "                     │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  核心思想：每条边都是两个区域的"公共边界"                                      │
│                                                                                │
│  就像一面墙，隔开了两个房间——这面墙既属于左边房间，也属于右边房间。            │
│                                                                                │
│  ┌─────────────────────────────────────────────────────────────────────────┐  │
│  │                                                                         │  │
│  │         这条边                                                          │  │
│  │           ↓                                                             │  │
│  │      ●────●────●                                                        │  │
│  │      │ 区域A │ 区域B │                                                  │  │
│  │      │       │       │                                                  │  │
│  │      ●───────●───────●                                                  │  │
│  │              ↑                                                          │  │
│  │         这条边被 A 和 B 两个区域共享                                    │  │
│  │         所以在统计"区域边数"时，这条边会被数2次！                       │  │
│  │                                                                         │  │
│  └─────────────────────────────────────────────────────────────────────────┘  │
│                                                                                │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  用简单例子验证：一个正方形                                                    │
│                                                                                │
│       ●───────●                                                               │
│       │       │          边数 e = 4（上、下、左、右 4条边）                   │
│       │ 区域1 │                                                               │
│       │       │          区域 f = 2（内部区域1 + 外部区域2）                  │
│       ●───────●                                                               │
│         区域2                                                                  │
│        (外部)                                                                  │
│                                                                                │
│  统计每个区域的边数：                                                          │
│  • 区域1（内部）：上边 + 下边 + 左边 + 右边 = 4条边                           │
│  • 区域2（外部）：上边 + 下边 + 左边 + 右边 = 4条边（从外面看也是4条）        │
│                                                                                │
│  ∑边数(f) = 4 + 4 = 8                                                         │
│                                                                                │
│  2e = 2 × 4 = 8                                                                │
│                                                                                │
│  验证：∑边数(f) = 8 = 2e  ✓                                                   │
│                                                                                │
│  ┌─────────────────────────────────────────────────────────────────────────┐  │
│  │  为什么相等？                                                           │  │
│  │                                                                         │  │
│  │  因为每条边恰好被2个区域共享：                                          │  │
│  │  • 如果边在图的"内部"，它分隔两个有限区域                              │  │
│  │  • 如果边在图的"边界"，它分隔一个有限区域和外部无限区域                │  │
│  │                                                                         │  │
│  │  所以：统计所有区域的边数时，每条边正好被数2次                          │  │
│  │        即：∑边数(f) = 2e                                               │  │
│  └─────────────────────────────────────────────────────────────────────────┘  │
│                                                                                │
└────────────────────────────────────────────────────────────────────────────────┘
```

**🔑 Step 3 详解：如何从两个式子得到 2e ≥ 3f ？**

```
┌────────────────────────────────────────────────────────────────────────────────┐
│                    把 两 个 式 子 结 合 起 来                                  │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  我们现在有两个关于 ∑边数(f) 的式子：                                          │
│                                                                                │
│  式子1：∑边数(f) ≥ 3f      （每个区域至少3条边）                              │
│  式子2：∑边数(f) = 2e      （每条边被2个区域共享）                            │
│                                                                                │
│  ┌─────────────────────────────────────────────────────────────────────────┐  │
│  │                                                                         │  │
│  │  因为 ∑边数(f) 同时满足两个条件：                                       │  │
│  │                                                                         │  │
│  │      ∑边数(f) = 2e     （等于 2e）                                      │  │
│  │      ∑边数(f) ≥ 3f     （大于等于 3f）                                  │  │
│  │                                                                         │  │
│  │  把"= 2e"代入"≥ 3f"：                                                  │  │
│  │                                                                         │  │
│  │      2e ≥ 3f                                                            │  │
│  │                                                                         │  │
│  │  两边同除以3：                                                          │  │
│  │                                                                         │  │
│  │      f ≤ 2e/3                                                           │  │
│  │                                                                         │  │
│  └─────────────────────────────────────────────────────────────────────────┘  │
│                                                                                │
│  用之前的例子验证：                                                            │
│                                                                                │
│      e = 8,  f = 5                                                             │
│                                                                                │
│      2e = 16,  3f = 15                                                         │
│                                                                                │
│      16 ≥ 15  ✓                                                               │
│                                                                                │
│      f = 5 ≤ 2×8/3 = 16/3 ≈ 5.33  ✓                                          │
│                                                                                │
└────────────────────────────────────────────────────────────────────────────────┘
```

#### 8.5.6 另一个例子：证明 K₃,₃ 不是平面图
