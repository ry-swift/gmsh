## 1. 什么是对偶？

### 1.1 生活中的对偶例子

**对偶的核心思想**：把一个空间中的对象，按照某种规则，映射到另一个空间。

| 原空间 | 对偶空间 | 映射规则 |
|--------|----------|----------|
| 实际地形（3D） | 地图（2D） | 等高线表示高度 |
| 音乐（声音） | 乐谱（符号） | 音高→位置，时长→符号 |
| 多面体 | 高斯球 | 法线方向映射 |

### 1.2 对偶的好处

```
原空间中困难的问题  ──映射──>  对偶空间中简单的问题
       ↑                              ↓
       └────────── 解答映射回来 ───────┘
```

## 2. 高斯球（Gaussian Sphere）

### 2.1 定义

**高斯球** = 半径为1的单位球，球心在原点。

### 2.2 高斯球的3D示意图

```
                    z轴 (向上)
                      ↑
                      │    北极 (0,0,1)
                      │   ●
                     ╱│╲ ╱ ╲
                   ╱  │  ╲   ╲
                 ╱    │    ╲   ╲
               ╱      │      ╲   ╲
        y轴 ←────────●────────→ x轴
              ╲       │O      ╱
                ╲     │     ╱
                  ╲   │   ╱
                    ╲ │ ╱
                      ●
                    南极 (0,0,-1)

        半径 = 1 (单位球)
```

### 2.3 关键理解：高斯球表示方向

球面上的每个点 = 一个方向（单位向量）

```
          ● 向上 (0,0,1)
         /│\
        / │ \
       /  │  \
      ●───●───●  ← 赤道表示所有水平方向
     向左  │  向右
           │
           ● 向下 (0,0,-1)
```

## 3. 多面体与高斯球的对偶关系

### 3.1 什么是法线？

**法线** = 垂直于平面、指向外侧的方向

```
        法线 n
           ↑
           │
    ╔══════╪══════╗
    ║      │      ║
    ║   平面      ║
    ║             ║
    ╚═════════════╝
```

**重要**：平行的平面，法线方向相同 → 在高斯球上对应同一个点！

### 3.2 对偶映射的三条规则

#### 规则1：面 → 点

```
   多面体的一个面              高斯球上的一个点

        /\                         ○
       /  \                       / \
      /____\      ══════>        (  ●  )   ← 点 = 面的法线方向
        ↑                         \ /
      法线 n                        ○
```

#### 规则2：边 → 弧

```
   两个面共享一条边            高斯球上连接两点的弧

      /\  /\                       E
     /  \/  \                     ╱
    / 面1 面2\     ══════>       ●
   /____边____\                   ╲
        ↑↗                         F
    两个法线                       ↑
                              弧 EF (测地线)
```

#### 规则3：顶点 → 区域

```
   多个面交于一个顶点          高斯球上多条弧围成的区域

         A                        ●E
        /|\                      /|\
       / | \                    / | \
      /  |  \      ══════>     ●──●──●
     面1 面2 面3               F  区域  G
                                  ↑
                            顶点A对应的区域
```

### 3.3 对偶映射汇总表

| 多面体（原空间） | 高斯球（对偶空间） |
|------------------|---------------------|
| 面 (Face) | 点 (Point) |
| 边 (Edge) | 弧 (Arc) |
| 顶点 (Vertex) | 区域 (Region) |

**欧拉公式在对偶中保持不变**：

- 多面体：V - E + F = 2
- 高斯球：R - A + P = 2（区域数 - 弧数 + 点数 = 2）

## 4. 四面体对偶映射示例

### 4.1 四面体的结构

```
              A
             /|\
            / | \
           /  |  \
          /   |   \
         /    |    \
        D─────┼─────B
         \    |    /
          \   |   /
           \  |  /
            \ | /
             \|/
              C
```

**统计**：
- 4 个顶点：A, B, C, D
- 6 条边：AB, AC, AD, BC, BD, CD
- 4 个面：ABC, ABD, ACD, BCD

### 4.2 每个面的法线方向

```
              A
             /|\
    n_ABD ← / | \ → n_ABC
           /  |  \
          /   |↓n_BCD
         D────┼────B
    n_ACD →\  |  /
            \ | /
             \|/
              C
```

4个面 → 4个法线 → 高斯球上4个点

### 4.3 对偶结果：球面上的四面体

```
    四面体                    高斯球上的对偶图

       A                           G (n_ACD)
      /|\                         /|\
     / | \                       / | \
    /  |  \     ════════>       /  |  \
   D───┼───B                   F───┼───E
    \  |  /                   (n_ABD) (n_ABC)
     \ | /                      \  |  /
      \|/                        \ | /
       C                          \|/
                                   H (n_BCD)
```

**对应关系**：

| 面 → 点 | 边 → 弧 | 顶点 → 区域 |
|---------|---------|-------------|
| ABC → E | AB → EF | A → 区域EFG |
| ABD → F | AC → EG | B → 区域EFH |
| ACD → G | AD → FG | C → 区域EGH |
| BCD → H | BC → EH | D → 区域FGH |
|         | BD → FH | |
|         | CD → GH | |

## 5. 曲面的对偶

### 5.1 三种曲面的对偶对比

```
┌─────────────┬───────────────────┬─────────────────────┐
│   曲面类型   │     法线特点       │   高斯球上的对偶     │
├─────────────┼───────────────────┼─────────────────────┤
│             │                   │         ●           │
│   平面      │ 所有点法线相同     │    一个点           │
│             │                   │                     │
├─────────────┼───────────────────┼─────────────────────┤
│    ┃ ┃      │                   │       ╭───╮         │
│    ┃ ┃      │ 法线水平旋转      │      (●●●●)         │
│   圆柱面    │ 绕一周            │       ╰───╯         │
│    ┃ ┃      │                   │    一个大圆(赤道)    │
├─────────────┼───────────────────┼─────────────────────┤
│     /\      │                   │        ╭─╮          │
│    /  \     │ 法线有固定        │       ( ● )         │
│   圆锥面    │ 倾斜角度          │        ╰─╯          │
│  /____\    │                   │    一个小圆          │
└─────────────┴───────────────────┴─────────────────────┘
```

### 5.2 为什么会这样？

```
平面：所有点的法线方向相同
      ═══════════
          ↑↑↑↑↑     → 高斯球上一个点

圆柱面：法线水平旋转360°
      │→│→│→│
      │ │ │ │       → 高斯球赤道一圈（大圆）
      │→│→│→│

圆锥面：法线有固定倾角，绕轴旋转
      ↗ ↑ ↖
       \|/          → 高斯球上纬度圈（小圆）
```

## 6. 实际应用：点云分割

### 6.1 问题背景

3D扫描得到点云 → 需要识别不同的几何面

### 6.2 利用对偶进行分割

```
Step 1: 计算每个点的法线
Step 2: 把法线映射到高斯球
Step 3: 在高斯球上进行聚类

    点云数据                    高斯球上的法线分布

  . . . . . . .                      ●●● ← 平面1
  . ┌─────┐ . .                     ╱   ╲
  . │ . . │ . .    ══════>        ●     ● ← 圆柱面
  . │ . . │ . .                    ╲   ╱
  . └─────┘ . .                      ●●● ← 平面2
  . . . . . . .
```

### 6.3 对偶的威力

| 原空间（点云） | 对偶空间（高斯球） |
|----------------|---------------------|
| 点混在一起，难区分 | 法线分布有明显模式 |
| 平面上的点 | 聚成一个点 |
| 圆柱面上的点 | 排列成大圆 |
| 圆锥面上的点 | 排列成小圆 |

## 附录：对偶的数学直觉

### 为什么"面"对应"点"？

```
面的唯一特征 = 法线方向
不管面在哪里、有多大，只要朝向相同，法线就相同
∴ 一个面 → 一个方向 → 高斯球上一个点
```

### 为什么"边"对应"弧"？

```
边 = 两个面的交界
两个面有两个法线 → 高斯球上两个点
沿边"转动"时，法线从 n₁ 连续变到 n₂
这个变化轨迹 = 球面上的弧（测地线）

    面1的法线 n₁
         ↗                    n₁●
   ╱─────╲                       ╲
  ╱ 面1   ╲   边                  ●n₂
 ╱─────────╲                     ↑
╱   面2     ╲               弧 n₁n₂
        ↘
    面2的法线 n₂
```

### 为什么"顶点"对应"区域"？

```
顶点 = 多个面的交点
这些面的法线 → 高斯球上多个点
相邻面之间的边 → 连接这些点的弧
这些弧围成的封闭区域 = 顶点对应的区域
```
