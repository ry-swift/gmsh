## 简介

本文档是一份全面、详细的Gmsh中文教学资料，旨在帮助您从零开始掌握这款强大的有限元网格生成工具。无论您是初学者还是有经验的用户，都能在这里找到所需的内容。

## 什么是 Gmsh？

Gmsh 是一款开源的三维有限元网格生成器，具备以下核心功能：

- **几何建模 (Geometry)**: 内置CAD引擎和OpenCASCADE内核，支持参数化建模
- **网格生成 (Mesh)**: 支持1D/2D/3D结构化和非结构化网格
- **求解器框架 (Solver)**: 内置有限元求解器框架
- **后处理 (Post-processing)**: 强大的数据可视化和分析功能

## 环境要求

- **Python**: 3.8+ (推荐 3.10+)
- **Gmsh**: 4.14.0
- **操作系统**: Windows / macOS / Linux

## 快速安装

```bash
pip install --upgrade gmsh
```

## 验证安装

```python
import gmsh
gmsh.initialize()
print(f"Gmsh API版本: {gmsh.GMSH_API_VERSION}")
gmsh.finalize()
```

## 代码约定

本文档中的所有代码示例遵循以下约定：

```python
import gmsh

# 初始化Gmsh（必须第一步调用）
gmsh.initialize()

# 创建新模型
gmsh.model.add("model_name")

# ... 几何建模和网格生成 ...

# 保存结果
gmsh.write("output.msh")

# 可选：启动GUI查看结果
# gmsh.fltk.run()

# 清理资源（必须最后调用）
gmsh.finalize()
```
